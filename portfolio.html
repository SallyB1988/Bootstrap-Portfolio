Liri Project

This project uses NodeJs. All commands and ouput are shown in a terminal window. It is a text based knock-off of Siri which can be used to find information about songs, movies or concerts. The options are:

concert-this

spotify-this-song

movie-this

run-batch-file

concert-this will find current concert information (from the Bands in Town Artist Events API) about whatever band/artist is supplied

spotify-this-song will find information (from the Spotify API) about whatever song is supplied

movie-this will find movie information (from the OMDB API) about whatever movie is supplied

run-batch-file will run all of the search commands that are stored in a file named search.bat

Terminal Version of Hangman Game

This project uses NodeJs and several JavaScript constructors. All interaction is done in the terminal window. There are three phrases to try to discover. Guess letters - if you are correct, all occurrances of the letter are filled in. If you are wrong, you are told how many incorrect guesses you have remaining.
