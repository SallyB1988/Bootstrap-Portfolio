This project uses data obtained from the ReliefWeb API. The data consists of recent worldwide disasters. The user can search for information about various types of disasters in countries around the world. The data can be filtered by country, disaster type, and year it occurred. There are two pie charts displaying the number of searches done for different countries and different types of disasters. These charts are updated every time a new search is entered.

Technologies Used

Handlebars Express NodeJS MySQL Sequelize JavaScript HTML JQuery CSS
